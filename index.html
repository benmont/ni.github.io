Welcome to the National Instruments GitHub

The @ni organization hosts a number of repositories used with National Instruments products, in two general categories:

  1. Open source software used to create the NI Linux Real-Time distribution, which is supported through https://decibel.ni.com/content/groups/ni-linux-real-time, and
  2. NI-specific projects where we are collaborating with our users on GitHub, which are supported through https://decibel.ni.com/content/groups/community-source-of-ni-code and work in conjunction with the @ni-dev GitHub organization.

Please refer to the individual repositories for project-specific details.

NI engineers with questions about GitHub can refer to the internal FAQ.
